# Technology Stack: FIT-O-CHARITY Dashboard\n\n## Core Languages & Runtimes\n- **TypeScript:** Primary programming language for both frontend and backend to ensure type safety and maintainability.\n- **Node.js (v18+):** Runtime environment for the backend service.\n\n## Frontend Architecture\n- **React 18:** Functional components with Hooks for UI logic.\n- **Vite:** Build tool and development server for rapid frontend iteration.\n- **Tailwind CSS:** Utility-first CSS framework for styling.\n- **Framer Motion:** Animation library for smooth transitions and dopamine-inducing UI effects.\n- **Lucide React:** Icon set for industrial visual elements.\n- **Recharts:** Data visualization for activity metrics.\n\n## Backend Architecture\n- **Express.js:** Web framework for RESTful API development.\n- **MongoDB (Mongoose):** NoSQL database for flexible participant and activity data storage.\n- **WebSockets (ws):** For real-time "Live" dashboard updates.\n- **Node-Cron:** For scheduled daily reminder jobs.\n- **Zod:** Schema validation for API requests and data integrity.\n\n## External Integrations\n- **Twilio SDK:** For sending WhatsApp reminders.\n- **Nodemailer:** For sending automated email notifications.
